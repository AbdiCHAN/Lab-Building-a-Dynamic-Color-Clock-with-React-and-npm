# Dynamic Color Clock with React

A **React app** that displays a **dynamic color clock** using the **date-fns** library.  
The clock shows the current date and time and updates in real-time with a dynamic color effect based on the current time.

---

## Features

- Real-time updating clock
- Dynamic background color based on current time
- Display of current date and time
- Built with React and date-fns
- Responsive design for different screen sizes

## How to Run Locally

Follow these steps to run the project on your machine:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/AbdiCHAN/Lab-Building-a-Dynamic-Color-Clock-with-React-and-npm.git

2. **Navigate into the project folder:**
cd Lab-Building-a-Dynamic-Color-Clock-with-React-and-npm

3. **Install Dependencies**
npm install

4. **Start the development Server**
npm start

5.Open your browser and go to http://localhost:3000
 to see the dynamic color clock.
